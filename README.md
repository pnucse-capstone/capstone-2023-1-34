# Capstone-2023-1-34


### 1. 프로젝트 소개
Chaining은 부산 지역 내 기부를 할 수 있는 이더리움 Dapp 플랫폼입니다. 블록체인 기술 및 스마트 컨트랙트를 활용하여 기부의 투명성과 신뢰성을 보장하고자 합니다. 기부 증서를 NFT를 활용한 토큰으로 발행하여 기부자의 기부 내역을 증명하고, 기부 횟수에 따라 플랫폼에서 만든 NFT를 지급하여 활동적인 기부 문화를 조성하도록 하였습니다.
